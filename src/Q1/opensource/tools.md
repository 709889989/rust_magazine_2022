# 其他工具

## lapce 发布 0.0.9 版本

lapce 是一款受 Xi-Editor 启发的编辑器，类似于 vscode。作者声称，他对 lapce 抱有雄心壮志，希望有朝一日可以和 VScode 竞争。

目标：

- 启动迅速，基于 Rust + Druid 开发，使用 wgpu 渲染。
- 支持远程开发
- 支持 LSP，支持代码智能提示、诊断等
- Vim 模式支持
- WASI 插件系统
- 内置终端

目前为 0.0.9 版本，看起来还不太完善，可以先关注。
